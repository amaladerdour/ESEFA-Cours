Le citron

Définition:
est un agrume, fruit du citronnier (Citrus limon), qui existe sous deux formes : le citron doux, fruit décoratif de cultivars à jus peu ou pas acide néanmoins classés Citrus limon (L.) Burm. f. (classification de Tanaka) et le citron acide, le plus commun de nos jours dont le jus a unpH d'environ 2,5

Utilisation du citron
Pour l'alimentation Pour la santé
  • Le jus frais est utilisé en assaisonnements sur le poisson
  • Il remplace le vinaigre dans les salades
